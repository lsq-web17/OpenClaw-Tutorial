## 📦 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/yourusername/OpenClaw-Tutorial.git
cd OpenClaw-Tutorial
```

### 2. 安装OpenClaw
```bash
# 使用官方安装脚本
curl -fsSL https://openclaw.ai/install.sh | bash
```

### 3. 运行第一个示例
```bash
cd examples/hello-world
python hello_skill.py
```
