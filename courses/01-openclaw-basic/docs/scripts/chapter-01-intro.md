### 第三部分：编写技能代码（10分钟）

**【画面**: 代码编辑器界面】
**旁白**:
"创建 `skill.py` 文件，编写我们的Hello World技能："

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
Hello World 技能
最简单的OpenClaw技能示例
"""

from datetime import datetime


class HelloSkill:
    """Hello World技能类"""
    
    def __init__(self):
        """初始化技能"""
        self.greeting = "👋 你好！我是你的OpenClaw助手！"
    
    def handle_message(self, message):
        """处理消息的核心函数"""
        # 转换为小写方便匹配
        msg = message.lower().strip()
        
        # 问候类消息
        if any(word in msg for word in ['hello', '你好', 'hi', '嗨']):
            return self.greeting
        
        # 时间类消息
        if any(word in msg for word in ['time', '时间', '几点', '现在']):
            current_time = datetime.now().strftime('%Y-%m-%d %H:%M:%S')
            return f"🕐 当前时间：{current_time}"
        
        # 默认回复
        return "🤔 试试说'hello'或'时间'？"


# 测试代码
if __name__ == "__main__":
    skill = HelloSkill()
    print("测试Hello World技能：")
    print("1. 输入'hello':", skill.handle_message("hello"))
    print("2. 输入'时间':", skill.handle_message("时间"))
    print("3. 输入'其他':", skill.handle_message("其他内容"))
```

**旁白**:
"代码解析：
1. `__init__` 方法：初始化技能
2. `handle_message` 方法：处理用户消息
3. 消息匹配逻辑：根据关键词返回不同响应
4. 测试代码：验证技能功能"
